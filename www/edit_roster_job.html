Note -
If you chose "per Kindy-wide" and enter "1" volunteer this creates 1 job, with one volunteer.
If you chose "per Unit" and enter "1" volunteer this creates 2 jobs, with one volunteer for each Unit.
If you chose "per Group" and enter "1" volunteer this creates 4 jobs, with one volunteer for each Group.
For example, if you want to create a centre-wide job with 5 volunteers (eg "Holiday Gardens"), choose "per Kindy-wide" and enter "5" volunteers.
